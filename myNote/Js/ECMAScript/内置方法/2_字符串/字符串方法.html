字符串方法

标签：方法, 字符串

slice()
语法：
stringValue.slice(start,end);
功能：
截取字符串。
返回值：
返回截取到的字符串
参数说明：
1、start：必须，指定子字符串的开始位置。
2、end：可选，表示子字符串到哪里结束，end本身不在截取范围之内，省略时截取至字符串的末尾。end如果超过字符串的长度，那么浏览器会默认传入的参数是字符串的长度+1。
3、当参数为负数时，会将传入的负值与字符串的长度相加。
let a = '012 210';
a.slice(2,5); // "2 2"; <= end不在截取的范围之内
a.slice(-5,-2); // "2 2" <= 等价于a.slice(2,5);

substring()
语法：
stringObject(start,end);
功能：
截取字符串。
返回值：
返回截取到的字符串
参数说明：
1、start：必须，指定子字符串的开始位置。
2、end：可选，表示子字符串到哪里结束，end本身不在截取范围之内，省略时截取至字符串的末尾。end如果超过字符串的长度，那么浏览器会默认传入的参数是字符串的长度+1。
3、当参数为复数时，自动将参数转换为0。
4、substring()会将较小的数作为开始位置，将较大的数作为结束位置。
let a = '012 210';
a.substring(0,2); // <= "01"
a.substring(-10,2); // <= "01"
a.substring(0,100); // <= "012 210"
a.substring(2,-1); // <= "01"
a.substring(-2,-1); // <= ""

substr()
语法：
stringObject(start,len);
功能：
截取字符串
返回值：
返回截取到的字符串
说明：
1、start：必须，指定子字符串的开始位置。
2、len：可选，表示截取的字符总数，省略时截取至子字符串的末尾。
3、当start为负数时，会将传入的负值与字符串的长度相加。
4、当let为负数时，返回空字符串。
let a = '012 210';
a.substr(0,2); // "01"
a.substr(0,0); // ""
a.substr(0,-2); // ""
a.substr(-2,2); // "10"
a.substr(-2,-2); // ""

charAt()
语法：
stringObject.charAt(index)
功能：
返回stringObject中index位置的字符，默认返回第一位，找不到返回 "" 或者是 undefined
返回值：
String
let a = '135 896';
a.charAt(0); // 1
a.charAt(4); // 8
a.charAt(8); // ""
a.charAt(3); // " "
a[0]; // <= ECMAScript5出现的新用法，为了保证兼容不推荐使用
a[8]; // undefined
a[4]; // " "

charCodeAt()
语法：
stringObject.charCodeAt(index);
功能：
stringObject中index位置字符的字符编码，默认返回第一位，找不到返回NaN
返回值：
Number
let a = '012345 ';
a.charCodeAt(0); // 48
a.charCodeAt(8); // NaN
a.charCodeAt(6); // 32

indexOf()
语法：
stringObject.indexOf(index);
功能：
从下标0开始检索，遇到给定(index)的参数返回参数在数组中的下标，默认返回-1，当给定的参数是一个""/''/``时默认返回0
返回值：
Number，默认返回-1，空字符(没有空格的空字符串)串默认返回0
let a = '012345678 9';
a.indexOf(); // -1
a.indexOf( ); // -1
a.indexOf(''); // 0
a.indexOf(' '); // 9
a.indexOf('10'); // -1
a.indexOf(9); // 10
a.indexOf(10); // -1
a.indexOf();

lastIndexOf()
语法：
stringObject.lastIndexOf()
功能：
从数组末尾开始检索，遇到给定(index)的参数返回参数在数组中的下标，默认返回-1，当给定的参数是一个""/''/``时默认返回0
返回值：
Number
let a = '0123 3210';
a.lastIndexOf(4); // -1
a.lastIndexOf(' '); // 4
a.lastIndexOf(0); // 8
