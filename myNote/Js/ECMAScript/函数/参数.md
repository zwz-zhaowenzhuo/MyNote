## 参数

```
function fn(num1, num2) {
    return num1 + num2;
}
```

> 实参 == 形参\
  fn(1, 2);\
  num1 = 1\
  num2 = 2

> 实参 < 形参\
  fn(1);\
  num1 = 1\
  num2 = undefined

> 实参 > 形参\
  fn(1, 2, 3, 4, 5);\
  num1 = 1\
  num2 = 2\

> arguments(类数组/类似数组)\
  arguments不会垮函数\
  arguments和形参指向同一个**地址**\
  arguments不是Array类型，它是Object类型
```
function fn(...num) {
    for (var i in arguments) {
        console.log(arguments[i]);
    }
}
fn(1, 2, 3);
// 类数组\
{
   '0': 1,
   '1': 2,
   '2': 3,
   length: 3
}
```

> 类数组方法
```
arguments.callee(); // 指向函数本身
```
