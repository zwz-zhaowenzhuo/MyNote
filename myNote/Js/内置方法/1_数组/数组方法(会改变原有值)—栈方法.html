数组方法(会改变原有值)—栈方法

标签：数组, 栈方法

splice()
——↓splice()的删除操作↓——
语法：
arrayObject.splice(index,count);
功能：
删除从 index 处开始的零个或多个元素。
返回值：
从原始数组中删除的项组成的数组（如果没有删除任何项，则返回空数组）
说明：
count是删除的项目数量，如果设置为0，则不会删除项目。
如果不设置，则删除从index开始的所有值。
大白话说明：
index：从那开始删除
count：删除多少个 如果不设置删除从index开始的所有值。
let a = [0,1,2,3,4];
a.splice(1); // (4) [1,2,3,4]
↑从下标1开始删除↑
console.log(a); // (1) [0]

a = [0,1,2,3,4];
a.splice(1,3); //  (3) [1,2,3]
↑从下标1开始删除3个↑
console.log(a); // (2) [0,4]
——↑splice()的删除操作↑——
——↓splice()的插入操作↓——
语法：
arrayObject.splice(index,itemx1,...,itemX);
功能：
在指定位置插入值
参数：
Index:起始位置
0：要删除的项数
item1...itemX：要插入的项
返回值：
从原始数组中删除的项组成的数组。    （如果没有删除任何项，则返回空数组）
说明：
从那开始，删除0项数组元素，插入那些数组元素
let a = ['a','b'];
a.splice(0,0,0,1); // []
console.log(a); // (4) [0,1,'a','b']
a.splice(2,0,2); // []
console.log(a); // (5) [0,1,2,'a','b']
——↑splice()的插入操作↑——
——↓splice()的替换操作↓——
语法：
arrayObject.splice(index,itemx1,...,itemX);
功能：
在指定位置插入值，且同时删除任意数量的项
参数：
Index:起始位置
0：要删除的项数
item1...itemX：要插入的项
返回值：
从原始数组中删除的项（如果没有删除任何项，则返回空数组）
说明：
从那开始，删除0项数组元素，插入那些数组元素
let a = ['x','a','b'];
a.splice(0,1,0,1,2); // ["x"]
console.log(a); // (5) [0,1,2,"a","b"]

push()
语法:
arrayObject.push(添加的数组元素1,添加的数组元素2,...添加的数组元素n);
功能：
把其他的参数按顺序添加到 arrayObject 的尾部。
返回值：
把指定的值添加到数组后的新长度    Ps:会改变原有值
let arrayObject = [0,1,2,3];
arrayObject.push(4,5,6);
console.log(arrayObject);// (7) [0,1,2,3,4,5,6]

unshift():
语法：
arrayObject.unshift(newEle1,newEle2,...newEleN);
功能：
把其它的参数按顺序添加到 arrayObject 的头部。
返回值：
把指定的值添加到数组后的新长度    Ps:会改变原有值
let arrayObject = [2,3];
arrayObject.unshift(0,1);
console.log(arrayObject); // (4) [0,1,2,3]


shift()
语法：
arrayObject.shift();
功能：
删除 arrayObject 中的第一个元素
返回值：
被删除的那个元素    Ps:会改变原有值
let arrayObject = [-1,0,1];
arrayObject.shift();
console.log(arrayObject); // (6) [0,1]


pop()
语法：
arrayObject.pop();
功能：
删除 arrayObject 中的最后一个元素
返回值：
被删除的那个元素    Ps:会改变原有值
let arrayObject = [0,1,99];
arrayObject.pop();
console.log(arrayObject); // (2) [0,1]

reverse()
语法：
arrayObject.reverse();
功能：
用于颠倒数组中的元素顺序。
返回值：
数组。
let arrayObject = ["厉害了" ,"牛逼" ,"真能叭叭" ,"别吵吵了"];
arrayObject.reverse();
console.log(arrayObject); // (4) ["别吵吵了" ,"真能叭叭" ,"牛逼" ,"厉害了"]

sort()
语法：
arrayObject.sort(sortby);
功能：
用于对数组的元素进行排序。
返回值：
数组    Ps:改变原有值
说明：
1、即使数组中的每一项都是数值，sort()方法比较的也是字符串（ASCII）
Ps:字符出比大小，比的是ASCII码 先对比字符串的第一个字符 第一个字符对应的ASCII码值越小该字符串就越小 当两个字符串中 第一个字符 对应的ASCII码相等时 那么在对比第二个ASCII码 以此类推
let asc = [12,88,9,10];
asc.sort();
console.log(asc); // => (4)[10,12,88,9]
2、sort()方法可以接收一个比较函数作为参数
无参数：
let arrayObject = [12,88,9,10];
arrayObject.sort();
console.log(arrayObject); // (4) [10,12,88,9]

带参数：
升序：
arrayObject.sort((a, b) => a - b );
console.log(arrayObject); // => (4) [9, 10, 12, 88]
降序：
arrayObject.sort((a, b) => b - a );
console.log(arrayObject); // => (4) [88, 12, 10, 9];
无序(随机排序)：
arrayObject.sort(() => Math.random() - 0.5);
