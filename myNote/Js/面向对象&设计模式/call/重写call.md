# 重写call

    只要是个函数就有call方法（是个啥？）
    改变this的指向（干嘛用的？）
    参数：（怎么用？）
        多个参数
        第一个是修改this的
        第二个之后是实参
            如果有多个call，第一个参数一定是会被调用的
            如果有多个call，最后一个call的的第一个参数为调用的那个函数，第二个参数是this指向，之后才是实参
    注意：（有没有坑？）
        null，undefined，都属于window

    怎么改变this:
        只要让一个函数地址，等于某一个对象下的方法，this字然就变成了那个对象
