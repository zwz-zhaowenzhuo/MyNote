# 工厂模式

    工厂模式就是一个可以批量生产单例模式的思想
    存在的意义就是为了复用

    ```
        function fn(name, age) {
            let obj = { }; // 相当于原材料
            obj.name = name; // 加工
            obj.age = age; // 加工
            return obj; // 出厂
        }
        let obj0 = fn('zwz',21),
            obj1 = fn('zwz',22);
    ```
